-----------Crear un rol administrador con Postman--------------------------------
Ruta:
http://localhost:9596/administrador/registrarRol

El primer rol a registrar en Postman debe ser el administrador, ya que dentro del API especificamos que el usuario con ID 1 es administrador y puede acceder al Dashboard mediante el status 200 que se le asigne

Especificar en el body -> JSON, el siguiente texto:
{
    "nombre":"administrador",
    "descripcion":"Cuenta de administrador",
    "isCrear": true,
    "isActualizar":true,
    "isBorrar":true
}

--------------Crear un rol usuario----------------------------------------
Ruta:
http://localhost:9596/administrador/registrarRol

El segundo rol que debemos registrar es el del usuario, ya que enviaremos un status 201 que especificara que es un rol diferente al de administrador y solo tendra acceso a la pagina de inicio

Especificar en el body -> JSON, el siguiente texto:
{
    "nombre":"usuario",
    "descripcion":"Cuenta de usuario",
    "isCrear": false,
    "isActualizar":false,
    "isBorrar":false
}


--------------Crear un usuario Administrador en Postman-------------------------
Ruta:
http://localhost:9596/administrador/registrar

Especificar en el body -> JSON, el siguiente texto:

{
    "correo":"admin@gmail.com",
    "contrasenia":"1234",
    "estatus":true,
    "fechaRegistro":"2022-10-15",
    "fechaVigencia":"2022-10-15",
    "rol_id":1
}

Donde el rol_id debe conicidir con el ID del rol administrador de la tabla de roles, en este caso es 1 porque fue el primer rol registrado


--------------Crear un usuario en Postman-------------------------
Ruta:
http://localhost:9596/administrador/registrar

Especificar en el body -> JSON, el siguiente texto:

{
    "correo":"user@gmail.com",
    "contrasenia":"1234",
    "estatus":true,
    "fechaRegistro":"2022-10-15",
    "fechaVigencia":"2022-10-15",
    "rol_id":2
}

Donde el rol_id debe conicidir con el ID del rol usuario de la tabla de roles, en este caso es 2 porque fue el segundo rol registrado
Si colocamos rol 1 estamos especificando que es Administrador.
